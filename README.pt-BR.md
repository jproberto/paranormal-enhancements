# Aprimoramentos Paranormais do JP

Módulo **Foundry VTT** exclusivo para o sistema **[Ordem Paranormal](https://foundryvtt.com/packages/ordemparanormal)**.  
Este módulo adiciona melhorias de qualidade de vida para itens, tornando o jogo mais fluido e imersivo.

---

## Funcionalidades (Resumo)

- 🔫 **Controle de Munição** – consumo e recarga automáticos de munição.  
- 💥 **Dano Crítico Personalizado** – cálculo de dano crítico personalizado para armas.  
- 🔦 **Lanternas / Iluminação** – luzes que podem ser ligadas/desligadas com atualização no token.  
- 🔋 **Gerenciamento de Bateria** – teste e recarga de baterias de itens com rolagem progressiva de dados.  

> Para detalhes técnicos completos, exemplos de uso e melhorias planejadas, veja [FEATURES.pt-BR.md](FEATURES.pt-BR.md).

## Requisitos

- **Foundry VTT**: versão 13 ou superior  
- **Sistema Ordem Paranormal**  
- **[lib-wrapper](https://github.com/ruipin/fvtt-lib-wrapper)** (v1.12.0 ou superior)  

---

## Uso (Básico)

- Use armas e munição normalmente; a munição será consumida automaticamente.  
- Ao rolar ataques, o dano crítico utilizará o cálculo personalizado quando aplicável.  
- Ligue ou desligue lanternas e itens de iluminação; o token será atualizado automaticamente.  
- Teste baterias com rolagens de dados e recarregue quando necessário.
