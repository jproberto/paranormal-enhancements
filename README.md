# JP's Paranormal Enhancements

A **Foundry VTT** module exclusively for the **[Ordem Paranormal](https://foundryvtt.com/packages/ordemparanormal)** system.  
This module adds quality-of-life enhancements for items, making gameplay smoother and more immersive.

---

## Features (Summary)

- 🔫 **Ammo Tracking** – automatic ammo consumption and reload.  
- 💥 **Critical Damage Enhancement** – custom critical damage calculation for weapons.  
- 🔦 **Flashlights / Illumination** – toggleable lights with token updates.  
- 🔋 **Battery Management** – test and recharge item batteries with progressive dice rolls.  

> For full technical details, usage examples, and planned improvements, see [FEATURES.md](FEATURES.md).

## Requirements

- **Foundry VTT**: version 13 or higher  
- **Ordem Paranormal system**  
- **[lib-wrapper](https://github.com/ruipin/fvtt-lib-wrapper)** (v1.12.0 or higher)  

---

## Usage (Basic)

- Use ammo and ranged weapons normally; ammo will be automatically consumed.  
- Roll attacks with weapons; critical damage will use the custom calculation if enabled.  
- Toggle lights on items like flashlights; the token’s light updates automatically.  
- Test batteries with dice rolls, recharge when needed.
